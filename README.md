# Portfolio

Personal portfolio built with Next.js 15, TypeScript, and Tailwind CSS.

**Live**: [akshayaparida.vercel.app](https://akshayaparida.vercel.app)

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Vercel

## Setup

```bash
pnpm install
pnpm dev
```
